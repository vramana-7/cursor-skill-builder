# Day 6 GitHub Polish Checklist

Score each category from 1-5.

## Categories

1) Readme Quality
- clear value proposition
- accurate progress updates
- roadmap reflects current state

2) Repository Structure
- folders and file names are intuitive
- examples/prompts/evals are easy to navigate

3) Contribution Readiness
- contribution guide exists
- commit/PR expectations are clear
- beginner can get started quickly

4) Changelog Clarity
- milestone history is easy to follow
- updates match actual shipped work

5) Portfolio Readiness
- repo tells a clear growth story
- recent commits show consistent execution

## Total Score

- Max: 25
- 21-25: polished and portfolio-ready
- 16-20: strong, minor improvements helpful
- 11-15: usable, needs more consistency
- <=10: major polish pass needed
