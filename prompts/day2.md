# Day 2 - Reusable Prompt Framework (60 minutes)

Goal: Create one reusable prompt template you can use every week.

## What You Will Build

- A reusable planner prompt in `prompts/template.md`
- One filled Day 2 run in `examples/day2-output.md`
- A short note on what changed from Day 1

## Step 1 (20 min): Review Day 1 outputs

Open `examples/day1-output.md` and identify:
- what structure helped most
- what constraints improved quality
- what was still too generic

Write 2-3 bullets in your own words before moving on.

## Step 2 (25 min): Use the reusable template

Open `prompts/template.md`.

1) Replace bracketed placeholders with your context.
2) Run the prompt in Cursor Chat.
3) Save the response into `examples/day2-output.md`.

## Step 3 (10 min): Tighten and rerun

Improve one part of the template:
- output format
- constraints
- success criteria

Run once more and compare results.

## Step 4 (5 min): Commit and push

```bash
git add .
git commit -m "Add Day 2 reusable prompt framework and outputs"
git push
```
