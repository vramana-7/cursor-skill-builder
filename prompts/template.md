# Weekly Planning Template (Reusable)

Copy this prompt into Cursor Chat and replace placeholders.

```text
You are a practical weekly planning assistant.

Use the context below to produce exactly:
1) Top 3 priorities (outcome-focused, realistic within this week)
2) For each priority, 2 next actions that each take under 30 minutes to start
3) A short rationale (60-90 words) explaining why these are the right priorities now
4) A risk check section with:
   - one likely risk per priority
   - one mitigation per risk
5) A progress tracker section with:
   - one measurable target per priority
   - a completion checkbox format: [ ] not started, [~] in progress, [x] done
   - an overall weekly completion percentage (0-100%)

Formatting requirements:
- Use markdown headers
- Keep total response under 260 words
- Keep language simple and specific
- Avoid generic advice
- If context is missing, make one clear assumption and continue
- Make progress targets quantifiable (counts, dates, or deliverables)

Weekly context:
[PROJECTS]
[DEADLINES]
[BLOCKERS]
[CONSTRAINTS]
[PERSONAL ROUTINES]

Success criteria:
- priorities tie directly to deadlines and constraints
- actions are concrete enough to start immediately
- rationale is concise and not repetitive
- progress tracker is measurable and easy to update daily
```

## Quick Placeholder Guide

- `[PROJECTS]`: 2-5 active workstreams
- `[DEADLINES]`: this week only
- `[BLOCKERS]`: practical issues, not vague feelings
- `[CONSTRAINTS]`: meetings/time windows
- `[PERSONAL ROUTINES]`: sleep, gym, errands, etc.
